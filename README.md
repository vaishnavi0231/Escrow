# Stock Broker Client Web Dashboard

## Project Description
This is a simple Stock Broker Client Web Dashboard that allows multiple users to register, login, and subscribe to stock tickers.
The dashboard updates stock prices in real-time without page refresh using JavaScript.

## Features
- User Registration and Login using Email
- Supports multiple users
- Subscribe to supported stocks:
  - GOOG
  - TSLA
  - AMZN
  - META
  - NVDA
- Real-time stock price updates (randomly generated every second)
- Each user has independent subscriptions
- No page refresh required

## Technologies Used
- HTML
- CSS
- JavaScript
- Browser LocalStorage

## How to Run
1. Open the hosted Web Page Link
OR
2. Clone the repository and open `index.html` in a browser
